About Event

​Join us for a full-day AI hackathon in Singapore focused on building autonomous web agents that can browse, search, and extract real-time data from the open web with **TinyFish**!

​Chatbots are cool, but they're limited. We’re bringing the Singapore dev community together to build autonomous web agents — AI that can actually browse, search, and extract the messy parts of the open web in real-time. TinyFish gives your agents the eyes and ears they need to actually do things in production.

​**Partnering with OpenAI**

​We're excited to be **partnering with OpenAI** to fuel this hackathon. Participants will get direct access to OpenAI's latest models and Codex tools to help architect, debug, and ship complex agentic workflows. By combining OpenAI’s world-class reasoning with TinyFish’s web-layer infrastructure, you’ll have everything you need to build the next generation of autonomous agents.

**The Challenge**

​There is no fixed problem statement. We want you to solve a **hard, real-world problem** using the Open Web as your database.

​Whether it's automating deep research, performing high-scale extraction where traditional scrapers fail, or building "eyes and ears" for autonomous agents - if it operates on the live web, it’s fair game. We value technical complexity, utility, and agents that can handle the "messy" parts of the internet.

​**Need inspiration? You could explore:**

- ​**🔹 Complex Data Extraction** — Structure data from dynamic or anti-bot sources that traditional tools can’t touch.
    
- ​**🔹 Autonomous Research** — Build agents that navigate the deep web to synthesize findings in real-time.
    
- ​**🔹 Market & Social Intel** — Create "sentries" that monitor pricing, news, or sentiment signals across fragmented platforms.
    
- ​**🔹 Agentic Infrastructure** — Build the evals or navigation logic that helps agents operate more reliably on the web.
    

​**The only rule: Build something high-leverage that pushes the boundaries of the agentic web.**
